# game-matematika-interaktif
Media pembelajaran matematika interaktif berbasis Web (HTML, CSS, JS) sebagai materi Workshop FMIPA UNM. Konversi modern dari Macromedia Flash ActionScript 2.0.

Repository ini berisi kumpulan game edukasi matematika interaktif berbasis **HTML5, CSS, dan JavaScript (Vanilla)** yang dirancang sebagai materi pelatihan/workshop media pembelajaran interaktif bagi guru-guru matematika di jurusan Matematika FMIPA Universitas Negeri Makassar (UNM).

Aplikasi ini merupakan bentuk modernisasi dan konversi dari media pembelajaran *legacy* berbasis **Macromedia Flash MX 2004 (ActionScript 2.0)** ke standar teknologi web modern yang *cross-platform*, ringan, dan tanpa memerlukan instalasi aplikasi khusus.

---

## 🚀 Fitur & Modul Game

1. **Matrix Blitz Arena (MBA):** Game perhitungan determinan matriks $2 \times 2$ dengan batasan waktu countdown.
2. **X-Terminator:** Game penentuan pasangan pengali ($k_1$ & $k_2$) untuk mengeliminasi koefisien variabel $x$ pada SPLTV.
3. **Flip Ratio:** Game tantangan menyelesaikan soal perbandingan berbalik nilai dengan algoritma generator angka bulat.

---

## 🛠️ Teknologi yang Digunakan

- **HTML5:** Struktur dan elemen DOM media pembelajaran.
- **CSS3:** Flexbox & Grid layouting interaktif.
- **JavaScript (ES6):** Logika matematika, generator soal acak, timer interval, dan validasi input.
- **GitHub Pages:** Deployment & hosting statis.
